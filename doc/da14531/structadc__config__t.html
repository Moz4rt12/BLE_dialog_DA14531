◆ input

uint8_t adc_config_t::input

Input channel; possible values: adc_input_se_t for single ended mode or ( adc_input_se_t | adc_input_diff_t << 4) for differential mode, e.g. (ADC_INPUT_SE_P0_1 | (ADC_INPUT_DIFF_P0_2 << 4)) initializes the P01 as positive channel of ADC and P0_2 as negative channel of ADC in differential mode.
