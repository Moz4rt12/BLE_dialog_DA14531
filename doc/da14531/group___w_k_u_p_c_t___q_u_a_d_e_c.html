void quad_decoder_clear_irq ( void  )

Clears respective pending interrupt from Quadrature Decoder and NVIC and masks quad decoder interrupt.

Note
By default the Quad Decoder is enabled by the hardware on power up. The Quad Decoder has to be stopped and its pending interrupt has to be cleared, if the Quad Decoder is not used by the application.
