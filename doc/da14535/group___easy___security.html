Search the database to find the slot with the bond data that match. If remove_type is remove_this_entry, then remove this entry and update ext mem. ◆ app_easy_security_ral_op()

void app_easy_security_ral_op ( enum app_ral_operations  ral_op,
struct gap_ral_dev_info *  dev_info 
)

Request a Resolving List operation.

Constructs a message gapm_rslv_list_mgt_cmd with and sends it to lower layers. If operation is APP_CLEAR_RAL or APP_GET_RAL_SIZE, device info will not be copied in the command. If operation is APP_ADD_DEV_IN_RAL and we operate in controller privacy device mode, then a second command will be send to set the device in that mode. In all operations, apart from APP_NETWORK_MODE_RAL and APP_DEVICE_MODE_RAL, lower layers will respond with a gapm completion event. If operation is APP_GET_RAL_SIZE, lower layers will also respond with a GAPM_RAL_SIZE_IND. If operation is APP_GET_RAL_LOC_ADDR or APP_GET_RAL_PEER_ADDR, lower layers will also respond with a GAPM_RAL_SIZE_IND.
