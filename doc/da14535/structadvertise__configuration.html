◆ addr_src

uint8_t advertise_configuration::addr_src

Own BD address source of the device:

  • GAPM_STATIC_ADDR: Public or Random Static Address according to device address configuration
  • GAPM_GEN_RSLV_ADDR: Generated Resolvable Private Address
  • GAPM_GEN_NON_RSLV_ADDR: Generated non-Resolvable Private Address

◆ adv_filt_policy

uint8_t advertise_configuration::adv_filt_policy

Host information advertising data (GAPM_ADV_NON_CONN and GAPM_ADV_UNDIRECT)

  • ADV_ALLOW_SCAN_ANY_CON_ANY: Allow both scan and connection requests from anyone
  • ADV_ALLOW_SCAN_WLST_CON_ANY: Allow both scan req from White List devices only and connection req from anyone. Supported only in broadcaster role.
  • ADV_ALLOW_SCAN_ANY_CON_WLST: Allow both scan req from anyone and connection req from White List devices only. Supported only in broadcaster role.
  • ADV_ALLOW_SCAN_WLST_CON_WLST: Allow scan and connection requests from White List devices only. Supported only in broadcaster role.

uint8_t advertise_configuration::mode

Host information advertising data (GAPM_ADV_NON_CONN and GAPM_ADV_UNDIRECT) Advertising mode :

  • GAP_BROADCASTER_MODE: Broadcaster mode
