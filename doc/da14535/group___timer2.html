◆ timer2_pwm_freq_set()

void timer2_pwm_freq_set ( uint32_t  pwm_freq,
uint32_t  input_freq 
)

Sets Timer2 output frequency of the PWM signals to the desired one.

Note
Timer2 output PWM frequency (as for DA14585/DA14586, sys_clk is the ONLY Timer2 input clock source) may be within the following range:
  • [2 Hz, 16 kHz], if Timer2 input clock frequency is 32 kHz,
  • [123 Hz, 1 MHz], if Timer2 input clock frequency is 2 MHz,
  • [245 Hz, 2 MHz], if Timer2 input clock frequency is 4 MHz,
  • [489 Hz, 4 MHz], if Timer2 input clock frequency is 8 MHz,
  • [977 Hz, 8 MHz], if Timer2 input clock frequency is 16 MHz. In general, Timer2 output PWM frequency may be within the following range:
  • minimum = (Timer2 input clock frequency) / (2^14 - 1),
  • maximum = (Timer2 input clock frequency) / 2.
Parameters
[in]pwm_freqThe desired frequency of the PWM signals in Hz.
[in]input_freqThe Timer2 input clock frequency in Hz.
