528  * This translates to ~26usec for XTAL32K and ~57usec for RCX.
529  *
530  * Thus, LP_ISR_TIME_USEC - XTAL16M settling time = 3357 - 2900 = 457usec and 457 - 57 = 400usec

533  * In principle, this time can be reduced to be at least 2 low power cycles.

535  * If the remaining time is not enough for periph_init() to complete then an ASSERTION will hit in

537  * XTAL16M is fixed, the user should increase the LP_ISR_TIME_USEC (and, consequently, the
538  * LP_ISR_TIME_XTAL32_CYCLES).
539  */
