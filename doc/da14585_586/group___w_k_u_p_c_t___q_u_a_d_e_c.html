◆ wkupct_enable_irq()

void wkupct_enable_irq ( uint32_t  sel_pins,
uint32_t  pol_pins,
uint16_t  events_num,
uint16_t  deb_time 
)

Enable Wakeup IRQ.

Parameters
[in]sel_pinsSelect enabled inputs (0-disabled, 1-enabled) For DA14585
  • Bits 0-7 -> port 0(P00..P07)
  • Bits 8-13 -> port 1(P10..P15)
  • Bits 14-15 -> port 2(P28,P29)
  • Bits 16-23 -> port 2(P20..P27)
  • Bits 24-31 -> port 3(P30..P37) For DA14531 and DA14535
  • Bits 0-11 -> port 0(P00..P011)
[in]pol_pinsInputs' polarity (0-high, 1-low) For DA14585
  • Bits 0-7 -> port 0(P00..P07)
  • Bits 8-13 -> port 1(P10..P15)
  • Bits 14-15 -> port 2(P28,P29)
  • Bits 16-23 -> port 2(P20..P27)
  • Bits 24-31 -> port 3(P30..P37) For DA14531 and DA14535
  • Bits 0-11 -> port 0(P00..P011)
[in]events_numNumber of events before wakeup interrupt. Max 255.
[in]deb_timeDebouncing time. Max 0x3F (63 msec)
