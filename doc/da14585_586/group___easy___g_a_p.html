void app_easy_gap_set_data_packet_length ( uint8_t  conidx,
uint16_t  tx_octets,
uint16_t  tx_time 
)

Set maximum transmission packet size and maximum packet transmission time to be used for the connection. This command may be sent either from peripheral or central.

Parameters
[in]conidxConnection Id number.
[in]tx_octetsMaximum transmission packet size.
[in]tx_timeMaximum packet transmission time.
Warning
app_easy_gap_get_peer_features() function shall be called prior to calling this function, to determine whether peer device supports LE Data Packet Length Extension.
