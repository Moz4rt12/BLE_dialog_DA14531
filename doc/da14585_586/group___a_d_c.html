◆ adc_init()

void adc_init ( const adc_config_t *  cfg)

Initializes and enables ADC.

Note
In the case of DA14531, adc_init applies the mandatory ADC settings, when the ADC_INPUT_SE_TEMP_SENS input channel is to be used.
In the case of DA14585/DA14586, the PD_PER domain shall be functional prior to ADC initialization: SetBits16(PMU_CTRL_REG, PERIPH_SLEEP, 0); while (!(GetWord16(SYS_STAT_REG) & PER_IS_UP));
Parameters
[in]cfgThe ADC configuration.
Note
Initialization does not take place if cfg is NULL.
