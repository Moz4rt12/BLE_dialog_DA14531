◆ GPIO_set_pad_latch_en()

__STATIC_FORCEINLINE void GPIO_set_pad_latch_en ( bool  pad_en)

Controls the latch configuration of the GPIO pads.

Parameters
[in]pad_enWhen set to true, the mode and state of the pad registers will be passed directly to the respective pads. When set to false, the state of the pads will be retained. Any change on the mode and state of the pad registers will not affect the pads until PAD_LATCH_EN field is set to 1.
