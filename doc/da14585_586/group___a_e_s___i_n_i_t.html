AES encrypt/decrypt operation.

Parameters
[in]keyThe key data.
[in]key_lenThe key data length in bytes (should be 16).
[in]inThe input data block.
[in]in_lenThe input data block length.
[in]outThe output data block.
[in]out_lenThe output data block length.
[in]enc_decAES_ENCRYPT for encryption, AES_DECRYPT for decryption.
[in]aes_done_cbThe callback to be called at the end of each operation.
[in]ble_flagsUsed to specify whether the encryption/decryption will be performed synchronously or asynchronously (message based) also if ble_safe is specified in ble_flags rwip_schedule() will be called to avoid missing any BLE events.
Returns
0 if successful, -1 if userKey or key are NULL, -2 if AES task is busy, -3 if enc_dec not 0/1, -4 if key_len not 16.
