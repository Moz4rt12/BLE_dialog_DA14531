◆ app_fill_random_byte_array()

void app_fill_random_byte_array ( uint8_t *  dst,
uint8_t  key_size,
uint8_t  array_size 
)

Fills an array with random bytes (starting from the end of the array) Assuming an array of array_size and a dynamic key size, so that key_size = M*4+N: Calls co_rand_word() M times and appends the 4 bytes of each 32 bit return value to the output array Calls co_rand_word() once and appends the N most significant bytes of the 32 bit return value to the output array Fills the rest bytes of the array with zeroes.

Parameters
[in]*dstThe address of the array
[in]key_sizeNumber of bytes that shall be randomized
[in]array_sizeTotal size of the array
